# React Enterprise Template (Vite + React 18 + TypeScript + Redux Toolkit + Material UI)

## Features
- React 18 (Vite)
- TypeScript
- Redux Toolkit
- Material UI (MUI)
- Axios for API calls
- Feature-based folder structure

## Setup
1. Install dependencies:
   ```bash
   npm install
   ```
2. Run development server:
   ```bash
   npm run dev
   ```
3. Build:
   ```bash
   npm run build
   ```

## Notes
- This is a starter template intended for learning and quick prototyping.
